# Sales Performance Analysis - Power BI

## Project Overview

This project presents an interactive Power BI dashboard designed to analyze sales performance across customers, products, regions, and time. The solution transforms raw sales data into actionable business insights through data modeling, DAX calculations, and dynamic visualizations.

The project follows a star schema data model consisting of fact and dimension tables to enable efficient reporting and analysis.

---

## Objectives

- Analyze overall sales and profitability
- Monitor sales performance across regions
- Evaluate product and category performance
- Understand customer segment behavior
- Track sales trends over time
- Build an interactive Business Intelligence dashboard

---

## Dataset Information

### Fact Table
- FactSales

### Dimension Tables
- DimCustomer
- DimProduct

The dataset includes approximately:

- 5,000 Sales Transactions
- 500 Customers
- 200 Products

Key attributes include:

- Sales
- Profit
- Quantity
- Discount
- Order Date
- Ship Date
- Customer Segment
- Region
- Category
- Sub-Category
- Brand
- Ship Mode
